CSS Positions explained

position: static;

An element with position: static; is not positioned in any special way; it is always positioned according to the normal flow of the page. You cannot use the following css properties: top, right, bottom, and left

CSS element with static position

position: relative;

An element with position: relative; is positioned relative to its normal position. This means that you can use the top, right, bottom, and left css properties

position: fixed;

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled:
